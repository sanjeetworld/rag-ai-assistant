# RAG-Based AI Assistant 🤖

A Retrieval-Augmented Generation (RAG) system built using LangChain and FastAPI to answer questions from custom PDF data.

## 🚀 Features
- Multi-PDF support
- Semantic search using FAISS
- Context-aware responses
- FastAPI backend
- Source attribution

## 🛠️ Tech Stack
- Python
- LangChain
- FAISS
- OpenAI API
- FastAPI

## ⚙️ Setup

```bash
pip install -r requirements.txt
uvicorn app:app --reload
